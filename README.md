# Boilerplate Express API

Phase 2 boilerplate project with:

 - Express
 - body-parser
 - nodemon
 - SQLite3
 - Knex
 - Jest


## Install

```
npm install
npx knex migrate:latest
npx knex seed:run
npm run dev
```

Be sure to check out the other npm scripts too.
